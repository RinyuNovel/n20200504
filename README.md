# Same as you are

## あらすじ

自立に近づく為にコンビニバイトを始めた大学生、深井柊。
彼はどうしようもなく自己中だった。
そんな彼に手を差し伸べるバイト仲間やオーナー等。
主人公は、彼等の恩に報いることができるだろうか。

人間の感情にスポットを当てた、
恋愛/伝奇小説
